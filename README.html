Tips

SOLARSAT Repositorio listo para abrir en local y correr todo en el navegador: Sistema Solar 3D + Mapa 2D con ground-track y filtros por altitud/inclinación. Cómo ejecutar Solo abre index.html haciendo doble clic. Todo usa CDNs con HTTPS (no requiere servidor local). Para el mapa, se consultan TLEs desde CelesTrak por HTTPS. Si estás offline, verás demo con ISS precargada. Si “no cargan los satélites” (checklist) Bloqueo por conexión: ¿Estás sin internet o detrás de un firewall que bloquea celestrak.org? CORS/HTTPS: El repo usa fetch hacia HTTPS, que funciona desde file:// en Chrome/Edge. Si tu navegador bloquea, prueba abrir con Live Server (VSCode) o python -m http.server. Demasiados satélites: El filtro por defecto trae pocos objetos (ISS, HUBBLE, NOAA). Activa “Cargar Activos (lento)” para más, o mantén el conjunto pequeño. Reloj del sistema: Si tu hora local está fuera de rango, la traza puede verse desfasada. Ajusta el reloj. Errores de consola: Abre F12 y revisa. El panel superior mostrará el último error legible.
